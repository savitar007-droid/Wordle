# 🎯 Wordle Game

A Wordle-inspired game built as a programming project.  
The goal is to guess the correct word within a limited number of attempts, with feedback provided after each guess.

---

## 🧠 How the Game Works

- You need to guess a hidden word.
- After each guess, feedback is given:
  - ✅ Correct letter in the correct position
  - 🟡 Correct letter in the wrong position
  - ❌ Letter not present in the word
- You have a limited number of attempts to guess the word correctly.

---

- **Language:** Java  
- **Build Tool:** SpringBoot 
- **IDE:** IntelliJ IDEA / Eclipse
